Magic Numbers

Featured YouTube Videos

The 3 videos displayed on the homepage. Get video IDs from YouTube URLs (the part after "watch?v=").

How to get a video ID:
1. Go to your video on YouTube
2. Look at the URL: youtube.com/watch?v=ABC123xyz
3. Copy the part after "v=" (in this example: ABC123xyz)

Google Drive Integration

Enter the Google Drive folder IDs for each asset type. The folder ID is the long string in the folder's URL after folders/.

Art Reference Images

When generating art direction, the AI will fetch these images from Drive and study them before writing the DALL-E prompt. This ensures the output matches your brand. Add one folder/file ID per line.

How to find a folder or file ID: Open the folder or file in Google Drive on the web. The ID is the long string in the URL after folders/ or file/d/. Up to 3 images per folder and 4 specific files are fetched (8 total max).

Visual Style Analysis

Run this once after adding your reference images. GPT-4o will study them and save a detailed style description that gets injected into every future art direction prompt — more reliable than re-analyzing images each time.

OAuth2 credentials are set server-side in /opt/satt-platform/.env as GOOGLE_OAUTH_CLIENT_ID, GOOGLE_OAUTH_CLIENT_SECRET, and GOOGLE_OAUTH_REFRESH_TOKEN. Run python3 scripts/get_gdrive_token.py locally to generate these values.
